## Commands list
Document contains project's useful commands.

### Build project:
```console
make build
```

### Start project:
```console
make start
```

### Shutdown project:
```console
make stop
```

### Run tests:
```console
make test
```

### Run linter:
```console
make lint
```

### Make django migrations:
```console
make migrations
```

### Run django migrations:
```console
make migrate
```
